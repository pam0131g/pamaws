簡介


我們是位於台灣的小軟體開發公司，使用過的工具有1999：Delphi，2004:DotNet，2012：DotNet Web，2019：Asp.Net Core，2020：AWS。
本網站保留在2017設計之地球首頁，提醒我要愛地球。
本網站主要是以 DotNet Core MVC 結合 AWS 來開發，因為是展示網站，我們將安全控管降至最低，只要是註冊的使用者就有全部權限。
將網站、資料庫、照片、文件等放到 AWS 雲端，只要能上網就可使用系統，不受地域的限制。
DotNet Core MVC 讓電腦、手機、iOS 等設備都可以使用系統，不受設備的限制，表示系統隨時隨地都可使用。

【Logistics】網頁：Asp.Net Core 結合 AWS Elastic Beanstalk MySql S3 三大功能與 Youtube 之雲端物流系統。

簡介


我們是位於台灣的小軟體開發公司，使用過的工具有1999：Delphi，2004:DotNet，2012：DotNet Web，2019：Asp.Net Core，2020目前正在開發：AWS。
本網站保留在2017設計之地球首頁，來提醒我要愛地球。本網站主要是以 DotNet Core MVC 結合 AWS 來展示，因為是展示網站，我們取消了安全控管。

【S3】網頁：只顯示 pamaws 有關之 Bucket，而且只限定名稱為 www.pamaws.com 之 Bucket 才可上傳、下載、刪除、轉出查詢結果到 Excel 或 Pdf。 File List 網頁：可對檔案名稱做關鍵字搜尋，或查詢特定檔案類型，並且具有分頁功能，每頁10項。

【Logistics】網頁：Asp.Net Core 連結 AWS RDS MS-SQL 資料庫，不日上架。
